=================
Rhyme Haiker
=================

押韻俳句生成器

A rhyming haiku generator.


=================
Install
=================

You can install rhyme-haiker using:

::

    $ pip install rhyme-haiker


=================
Usage
=================

1. Prepare a Japanese text corpus.
2. Preprocessing.

   ::

       $ haiku prep hoge_corpus.txt

3. Compose.

   ::

       $ haiku compose
